# Projeto Fase 1 - Projetar Aplicações Baseadas em IA na Nuvem

## Descrição do Projeto

Este projeto é a primeira fase da disciplina **Projetar Aplicações Baseadas em IA na Nuvem**. Ele foi desenvolvido pelos alunos **Rafael Barbosa** e **Ildefonso Melo**, sob a orientação do professor **Matheus Batista**. O objetivo é criar uma aplicação simples baseada em IA para análise de sentimentos, utilizando FastAPI e hospedada na nuvem.

## Pré-requisitos

Antes de executar o projeto, certifique-se de ter os seguintes itens instalados em seu ambiente:

- Python 3.8 ou superior
- Gerenciador de pacotes pip
- Ambiente virtual (opcional, mas recomendado)
